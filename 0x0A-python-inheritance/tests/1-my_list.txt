=======================
How to use 1-my_list.py
=======================

``obj.print_sorted()`` prints out a sorted (ascending sort) list

::

	>>> MyList = __import__('1-my_list').MyList

	>>> MyList.print_sorted(1)
	Traceback (most recent call last):
		...
	TypeError: 'int' object is not iterable

	>>> MyList.print_sorted()
	Traceback (most recent call last):
		...
	TypeError: MyList.print_sorted() missing 1 required positional argument: 'self'

	>>> MyList.print_sorted("hello")
	['e', 'h', 'l', 'l', 'o']

	>>> MyList.print_sorted([1, 3, 4, 1, 5])
	[1, 1, 3, 4, 5]

	>>> MyList.print_sorted(["hello", "Stephen", "Seyram", "Reuben"])
	['Reuben', 'Seyram', 'Stephen', 'hello']

	>>> MyList.print_sorted([1, 3, 4, 3], [24, 3, 3, 2])
	Traceback (most recent call last):
		...
	TypeError: MyList.print_sorted() takes 1 positional argument but 2 were given

	>>> MyList.print_sorted([-1, -23, 3, -8, 100])
	[-23, -8, -1, 3, 100]

	>>> MyList.print_sorted([])
	[]
