=======================
How to use 1-my_list.py
=======================

``obj.print_sorted()`` prints out a sorted (ascending sort) list

::

	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
