===========================
How to Use 3-say_my_name.py
===========================

``say_my_name()`` returns the a string containing the first name
and last name combined together

::

	>>> say_my_name = __import__('3-say_my_name').say_my_name

	>>> say_my_name("Stephen", "Twum")
	My name is Stephen Twum

	>>> say_my_name("Seyram", "Attivor")
	My name is Seyram Attivor
	
	>>> say_my_name("Reuben")
	My name is Reuben 
	
	>>> say_my_name("33")
	My name is 33
